# Vicenta Studio - Website

Página web para Vicenta Studio con diseño minimalista moderno.

## Archivos Creados

- `index.html` - Página principal del estudio
- `tempera.html` - Página dedicada al juego Tempera
- `styles.css` - Estilos principales con estética minimalista
- `tempera.css` - Estilos específicos para la página de Tempera
- `vicenta.jpeg` - Logo del estudio

## Características del Diseño

### Estética Minimalista
- Paleta de colores oscura (negro/gris) con acentos blancos
- Tipografía moderna: Inter con diferentes pesos (300-700)
- Animaciones sutiles y transiciones suaves
- Diseño limpio y minimalista
- Efectos de hover elegantes
- Letter-spacing negativo para títulos (estilo moderno)

### Responsive Design
- Adaptado para desktop, tablet y móvil
- Navegación optimizada para diferentes pantallas
- Grid layouts que se ajustan automáticamente

## Cómo Usar

1. Abre `index.html` en tu navegador para ver la página principal
2. Navega a Tempera haciendo click en "Descubre más" o abre `tempera.html` directamente

## Personalización

Para cambiar los colores, edita las variables CSS en `styles.css`:

```css
:root {
    --bg-dark: #0a0a0a;
    --bg-secondary: #1a1a1a;
    --text-primary: #ffffff;
    --text-secondary: #888888;
    --accent: #ffffff;
    --accent-hover: #e0e0e0;
}
```
